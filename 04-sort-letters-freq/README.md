# Sort Letters by Frequency
## Description

Write a python function named sort_letters_by_frequency that takes a string as an argument and returns a new string of the letters present in the input in descending order of their frequency. If two or more letters have the same frequency, they should appear in the output according to their usual order in the alphabet. You can assume the input is at least one letter long and contains only lowercase English letters.

### Notes
* Make sure that your function returns a string and not a list of characters.
